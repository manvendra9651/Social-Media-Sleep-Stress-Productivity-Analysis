# Social-Media-Sleep-Stress-Productivity-Analysis
Project Overview

This project analyzes the relationship between social media usage, sleep patterns, stress levels, exercise habits, and productivity using Python, Data Analysis, and Machine Learning techniques.

The goal of this project is to identify behavioral patterns and understand how digital lifestyle affects mental wellness and productivity.

Objectives
Analyze social media usage behavior
Study the impact of sleep on productivity
Detect stress-related patterns
Identify productivity factors
Build predictive machine learning models
Create interactive dashboards
Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Dataset Features

The dataset contains information about:

Daily Screen Time
Social Media Usage Hours
Sleep Hours

Study/Work Hours

Feature Engineering

Key Insights
Higher screen time increases stress levels
Better sleep improves productivity
Exercise helps reduce stress
Excessive social media usage negatively impacts productivity
